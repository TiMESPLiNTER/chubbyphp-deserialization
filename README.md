## Description

A simple deserializer.

## Requirements

 * php: ~7.0
 * psr/log: ~1.0

## Suggest

 * container-interop/container-interop: ~1.0
 * pimple/pimple: ~3.0

## Installation

Through [Composer](http://getcomposer.org) as [chubbyphp/chubbyphp-deserialize][1].

```sh
composer require chubbyphp/chubbyphp-deserialize "~1.0@dev"
```
